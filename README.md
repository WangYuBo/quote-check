本项目是一个专门为图书校对引用是否符合引用原文的软件。
例如：用户上传书稿A，书稿A第一章第一节第8页，有引用《道德经》一句“惚兮恍兮，其中有象”，并解释为“像‘道’一样模糊不清，但是其中有‘象’”。
用户上传了指定参考的原文图书或文章B，如指定了中华书局楼宇烈校王弼注本的《老子道德经》，那么就从B中查找是否有这句话、这句话文字是否准确无误A中这句话的解释是否与B一致、A中这句话与上下文是否相符；
输出校对结果要有原因。
如果书稿中有某些引用，但用户并未给出对应引用源，则提示用户导入引用源。

支持epub、txt、docx、markdown、pdf多格式；

### 三维度校对
- **文字准确性**：逐字比对引用与原文，识别文字、标点的增减改动
- **解释一致性**：核查作者解释是否准确反映原文意图，识别过度解读或曲解
- **上下文相符性**：评估引用是否恰当，检查断章取义或语境偏离
