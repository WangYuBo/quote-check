## Summary of the Quote Matching Pipeline

1. **Extraction**: LLM extracts quotes from the manuscript using the `extract` prompt
2. **Similarity Search**: `pg_trgm` searches reference paragraphs for passages similar to the quote text
3. **Pre-classification**: Based on trigram similarity thresholds (>=0.75 MATCH, >=0.4 PARTIAL_MATCH)
4. **LLM Verification**: The quote + matched reference passages are sent to the LLM with the `verify` prompt for three-dimensional assessment (text accuracy, interpretation, context)
5. **Final matchStatus**: The pre-classified status (from trigram similarity) takes precedence over the LLM verdict when references are uploaded
6. **Objective Confidence**: A weighted formula (refHit + locationValid + crossModel) computes confidence, explicitly discarding any LLM self-assessed scores
7. **Freeze**: Results are aggregated into a report snapshot, and the task is marked COMPLETED
