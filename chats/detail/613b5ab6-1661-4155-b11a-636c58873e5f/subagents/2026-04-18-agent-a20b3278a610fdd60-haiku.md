### 3. **Seven Constraints** (engineering-and-ethics-notes.md 12–105)

1. **Content Moderation Rejection** — Must explicitly mark rejections; never fake "pass" results
2. **Log Redaction** — Strip manuscript/reference original text from logs; hash + metadata only
3. **CJK Character Engineering** — OpenCC conversion (not `str.translate`); support Unicode CJK Ext B-G; handle composite characters (卅/廿) and variant forms
4. **Long Task Durability + Idempotency** — Persist task state to DB; support resume-on-restart; use idempotent keys for API calls
5. **Product Messaging** — Say "assistive tool" not "replace editor"; explicit "AI suggestion, please verify" UI text
6. **Forbid Single Score** — Display three dimensions independently; **never** calculate weighted average
7. **Report Immutability** — Freeze content + model + prompt + generation timestamp; cannot batch-update old reports; exports include version triple
